# ShakespeareGPT

A decoder-only transformer with character-level tokenization to generate Shakespeare-like text.

Inspired by [this video](https://youtu.be/kCc8FmEb1nY) from Andrej Karpathy.

Tiny Shakespeare dataset can be downloaded from [here](https://raw.githubusercontent.com/karpathy/char-rnn/master/data/tinyshakespeare/input.txt).
